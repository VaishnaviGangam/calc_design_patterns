#Project: Command Pattern and Plugins Homework 5
Made changes to the previous assignment to make into a interactive calculator

## Setup Instructions
1. Clone the repo
2. CD into the project folder
3. Create the virtual environment
4. Activate the virtual environment (VE)
5. Install Requirements

## Test Commands
1. pytest run all tests
2. Type 'menu' to show the menu.
3. Use any of the commands show in the menu followed by two numbers with a space in between. Eg.: add 134 1243
4. Type 'exit' to exit from the application
